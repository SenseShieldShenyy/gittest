DataArea : 读取文件，新增许可、rsa 私钥签名、写入数据区、生成D2C文件
程序使用
程序主要作用是分两次读取同一目录下的一个示例 exe，将其内容别写入两个许可的公开区，将其签名后的数据分别写入两个许可的只读区，并制作两个许可升级包
程序开头用宏定义固定了用户锁的锁号 lock_sn\ guo_pan,使用这需要根据自身手上的两把用户锁改锁号（事实上修改一个就可以了，只是用一个来判断数据写入的先后）
程序固定了许可ID为21号，使用者可以根据自己的需要修改
程序253行 判断通过锁号判断数据区的写入内容顺序


ReadPubData : 读取数据区数据，rsa公钥验签、写出并启动文件
程序运行时需要插入含有同样许可的加密锁，根据自己的许可修改代码处的许可ID
